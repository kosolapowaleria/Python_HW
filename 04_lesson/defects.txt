DEFECT ID: D001
НАЗВАНИЕ: Излишнее использование метода index() в методе класса contains():
ВАЖНОСТЬ: Средняя
СТАТУС: Открыт

Описание:

Метод 'contains' проверяет наличие символа через 'str.index' с исключением.

Такое использование:
- усложняет код;
- содержит избыточную логику(сравнение '> -1' всегда истинно при успехе)

Шаги:
1. Создать экземплят 'StringUtils'
2. Вызвать 'contains('python','x')'
3. Удостовериться, что код работает, но через исключение.

Фактический результат:
Код работает корректно, возвращает булев результат.

Ожидаемый результат и рекомендуемое исправление:
Использовать оператор 'in':
  return symbol in string -->

  def contains(self, string: str, symbol: str) -> bool:
    return symbol in string

СРЕДА:
  - Python 3.8+
  - Файл: 'string_utils.py'
  - ОС: Windows 11
